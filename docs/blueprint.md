# **App Name**: LocalFind

## Core Features:

- Service Browsing: Allow users to browse service providers by category.
- Provider Search: Enable searching for providers by keyword and location.
- Provider Profiles: Display detailed profiles for each service provider, including contact information, services offered, and reviews.

## Style Guidelines:

- Primary color: Fresh green (#6ACC8B) to evoke a sense of trust and local connection.
- Background color: Light, desaturated green (#EBF4EC). This creates a clean and unobtrusive background that ensures readability and enhances the focus on content.
- Accent color: Light blue (#84A7D9) to complement the green, providing a calm and reliable feeling for interactive elements.
- Font: 'PT Sans', a humanist sans-serif, for a modern look and readability.
- Clean and user-friendly layout with clear categorization and intuitive navigation.
- Use simple, recognizable icons to represent service categories and provider features.
